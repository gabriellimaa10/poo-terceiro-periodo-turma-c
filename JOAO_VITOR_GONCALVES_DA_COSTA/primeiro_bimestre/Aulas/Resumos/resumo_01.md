# O que é markdown

É uma liguagem de marcação simples e fácil, uma forma mais fácil de escrever html.

## Formatações

- **negrito** -> \*\* \*\* ou \_\_ \_\_

- _italico_ -> \* \* ou \_ \_

- **_negrito e italico_** -> **\* \*** **\_ \_**

\--- -> escreve uma linha para separar

\# -> usadas para criar tópicos

\``` -> usado para colocar blocos de códico

```
console.log("A")
```
